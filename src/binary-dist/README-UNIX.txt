

    Y A N E L
    =========


    Requirements and Prerequisites
    ------------------------------

    Java Development Kit version 1.6 or higher (e.g. http://java.sun.com)
    (NOTE: Please make sure to set the JAVA_HOME environment variable after installing JDK)


    Getting Started
    ---------------

    1) Start Yanel by executing ./start.sh
       IMPORTANT: Please make sure that there is no other server running on port 8080 (netstat -nt -l)
       
    2) Browse to http://localhost:8080/

    3) Stop Yanel by executing ./stop.sh


    More Information
    ----------------

    http://www.yanel.org
