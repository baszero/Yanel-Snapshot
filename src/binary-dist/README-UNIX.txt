

    Y A N E L
    =========


    Prerequisites
    -------------

    Java Development Kit version 1.5 or higher (http://java.sun.com, e.g. J2SE SDK v 1.5.0_15)
    (NOTE: Please make sure to set JAVA_HOME after installing JDK, e.g. export JAVA_HOME=/usr/local/jdk1.5.0_15)


    Getting Started
    ---------------

    1) Start Yanel by executing ./start.sh
       IMPORTANT: Please make sure that there is no other server running on port 8080 (netstat -nt -l)
       
    2) Browse to http://localhost:8080/

    3) Stop Yanel by executing ./stop.sh
