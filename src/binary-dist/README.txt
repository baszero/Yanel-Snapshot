

    Y A N E L (Yet Another Lenya)
    =============================


    Prerequisites
    -------------

    Java Development Kit version 1.4.2 or higher (http://java.sun.com, e.g. J2SE SDK v 1.4.2_12)


    Getting Started
    ---------------

    1) Startup Tomcat by executing ./apache-tomcat-5.5.20/bin/startup.sh
    2) Browse to http://localhost:8080/yanel/


    More Information
    ----------------

    http://yanel.wyona.org
