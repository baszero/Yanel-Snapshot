Node/File resource

Overview

Name:
file
Namespace:
http://www.wyona.org/yanel/resource/1.0
UniversalName
<{http://www.wyona.org/yanel/resource/1.0}file/>
Description:
Generic node resource
Java Class:
org.wyona.yanel.impl.resources.node.NodeResource

Examples

Specify mime type if suffix is not sufficient to guess mime type

    <yanel:resource-config xmlns:yanel="http://www.wyona.org/yanel/rti/1.0">
      <yanel:rti name="file" namespace="http://www.wyona.org/yanel/resource/1.0"/>

      <yanel:property name="mime-type" value="image/png"/>
    </yanel:resource-config>

Specify repository source path if different from URL (e.g. /foo/bar.html)

    <yanel:resource-config xmlns:yanel="http://www.wyona.org/yanel/rti/1.0">
      <yanel:rti name="file" namespace="http://www.wyona.org/yanel/resource/1.0"/>

      <yanel:property name="src" value="/foo/bar.xhtml"/>
      <yanel:property name="mime-type" value="application/xhtml+xml"/>
    </yanel:resource-config>
