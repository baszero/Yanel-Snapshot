Archive URL

Usecase Description

  • Enable Yanel toolbar
  • Click on "File" menu
  • Click on menu item "New"
  • Select "Archive URL"
  • Enter some URL, e.g. http://www.nyt.com
  • Enter new path, e.g. /my-local-nyt-frontpage-copy.html
  • Click "Save new resource"
