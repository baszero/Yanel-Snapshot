The structure of the 'Yanel website' data repository

(Last updated June, 2011 revision 58670)

  • The repository is split into three main directories:
    • data: Actual content
    • yarep-meta: Meta information
    • search-index: Lucene search index (fulltext and properties)
  • Inside the yarep-meta directory (1), one will find some additional information about the actual content (e.g. last modified, user who created/updated content)
  • The root of the data directory (2) contains the webpage index file (index.html), as well as other pages which have the same path as the URL referenced inside the Browser.
  • The CSS files are located inside the directory data/css (3).
  • The following files (4) are kind of special:
    • The sitetree.xml file contains the structure of the navigation inclusive their reference.
    • The translation.xml file associates the various URLs of the same content in different languages.
    • The introspection files provide the meta information for Neutron based editing tools (e.g. Yulup).
  • The yanel website is using XSLT in order to render pages dynamically. The most important XSLT file global.xsl is located in the xslt folder (5).
