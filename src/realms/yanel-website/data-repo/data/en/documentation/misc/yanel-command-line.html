Yanel command line (console)

One can also run Yanel on the command line as follows:

  • Edit: src/core/java/org/wyona/yanel/cmdl/YanelCommandLine.java
  • Build: ./build.sh compile-cmdl
  • Run: ./yanel.sh cmdl /yanel-website/en/documentation/misc/yanel-command-line.html?yanel.resource.viewid=source
  • Debug: tail -f logs/log4j-cmdl.log
