Acerca de Yanel

Todo es gestión de contenidos (WCMS, DMS, CRM, ERP, E-Mail, DRM, ...) y Yanel es un humilde intento de proporcionar una solución para todas sus necesidades de gestión de contenidos ofreciendo un Marco de Gestión de Contenidos escrito en Java y orientado a servicios de Código Abierto.

La comunidad de Yanel ha definido una serie de principios para sostener un desarrollo a largo plazo.

Principales Características

  • Marco de Recursos con interfaces publicadas versionadas lo que permite un desarrollo continuo
  • Soporta el protocolo Neutron y el Protocolo de Publicación Atom
  • Clusterable
  • Repositorios de Contenidos arbitrarios (ej: Sistema de Archivos, Subversion, BD Relacional, ...)
  • Separación de Contenidos y Aspecto
  • Múltiples portales
  • Soporte de Reverse Proxy
  • Multiplataforma (Continuamente testeado en Linux, Mac OS X, Windows)
