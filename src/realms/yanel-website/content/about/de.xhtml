Ueber Yanel

Alles ist Content Management (WCMS, DMS, CRM, ERP, PIM/E-Mail, DRM, ...) und Yanel ist ein bescheidener attempt to provide eine Loesung fuer alle Content Management Beduerfnisse by offering an Open Source service oriented Content Management Framework geschrieben in Java.

Die Yanel community hat eine Reihe von Prinzipien definiert um eine langfristige Entwicklung und einen entsprechenden Investitionsschutz zu garantieren.

Download

See the download page for information on obtaining releases, snapshots, source code, etc.

Features

  • Resource Framework mit versionierten Schnittstellen which allow continuous development
  • Supports the Neutron protocol
  • Clusterable
  • Arbitrary Content Repositories
  • Separation of Content and Layout
  • Multiple Sites
  • Reverse Proxy Support
  • Cross-Platform (Linux, Mac OS X, Windows (not tested yet), ...)
