Get the Source of Yanel

Use Subversion (a list of various SVN clients for Windows, Mac OS X, etc. can be found here) to retrieve the Yanel source from http://svn.wyona.com/repos/public/yanel/trunk: 

svn co http://svn.wyona.com/repos/public/yanel/trunk yanel-trunk

Username: anonymous
Password: anonymous

To get started with the source see the "Getting Started" guide or the README.txt.

Developer Access

svn co --username USERNAME https://svn.wyona.com/repos/public/yanel/trunk yanel-trunk
