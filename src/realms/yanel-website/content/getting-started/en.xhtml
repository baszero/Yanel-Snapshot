Getting Started

Prerequisites

Download and install Java Development Kit version 1.4.2 or higher (http://java.sun.com, e.g. J2SE SDK v 1.4.2_12)

(NOTE: After installing JDK, do set JAVA_HOME)

Download Yanel

Download Yanel source code.

Install Yanel

Install from source:

  • Start a new Command Shell "cmd" (Start - > Execute and enter "cmd")
  • Change into Yanel source directory (e.g. cd C:\....\yanel-trunk)
  • Run script "configure.bat"
  • Run script "build.bat"
  • Run script "yanel.bat start" to start Yanel as a server
  • (Run script "yanel.bat stop" to stop Yanel as a server)
