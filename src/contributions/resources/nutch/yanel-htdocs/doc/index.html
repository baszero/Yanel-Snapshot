Nutch resource

Overview

Name:
nutch
Namespace:
http://www.wyona.org/yanel/resource/1.0
Universal Name:
<{http://www.wyona.org/yanel/resource/1.0}nutch/>
Description:
Getting Search results from a Nutch crawl (tested nutch version 0.8.x)
Java Class:
org.wyona.yanel.impl.resources.NutchResource

Examples

Search results with some XSLT applied using the default configuration src/contributions/resources/nutch/conf/nutch-local.xml

    <yanel:resource-config xmlns:yanel="http://www.wyona.org/yanel/rti/1.0">
      <yanel:rti name="nutch" namespace="http://www.wyona.org/yanel/resource/1.0"/>

      <yanel:property name="global-xslt" value="/xslt/global.xsl"/>
    </yanel:resource-config>

Search results with some XSLT applied using a custom configuration

    <yanel:resource-config xmlns:yanel="http://www.wyona.org/yanel/rti/1.0">
      <yanel:rti name="nutch" namespace="http://www.wyona.org/yanel/resource/1.0"/>

      <yanel:property name="global-xslt" value="/xslt/global.xsl"/>
      <yanel:property name="nutch-config" value="file:/home/USER/nutch-config.xml"/>
    </yanel:resource-config>
