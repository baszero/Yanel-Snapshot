
   README
   ======

This resource-type exist in order to experiment with GWT features.

To use it

  - uncomment this resource-type within conf/resource-types.xml or add it to conf/local/local.resource-types.xml
  - compile it (build -f ./build.sh src/contributions/resources/GWT-sandbox/build.xml)
  - load the test page, e.g http://localhost:8080/yanel/yanel/resource-types/http://www.wyona.org/yanel/resource/1.0::gwt-trials/trials.html
