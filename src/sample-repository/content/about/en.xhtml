About

Everything is content management (WCMS, DMS, CRM, ERP, E-Mail, DRM, ...) and Yanel is a humble attempt to provide a solution for all your content management needs by offering an Open Source service oriented Content Management Framework written in Java.

The Yanel community has defined a set of principles to sustain long-term development.

Download

See the download page for information on obtaining releases, snapshots, etc.

Features

  • Resource Framework with versioned interfaces which allow continuous development
  • Supports the Neutron protocol and Atom Publishing Protocol
  • Clusterable
  • Arbitrary Content Repositories
  • Separation of Content and Layout
  • Multiple Sites
  • Reverse Proxy Support
  • Cross-Platform (Linux, Mac OS X, Windows (not tested yet), ...)
