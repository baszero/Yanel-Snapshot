How a new realm is being added

  • This document explains how to add a new realm to Yanel. We will explain this using the Yulup Website as an example. From now on, we will refer to the directories that contain Yanel and the Yulup website as $YANEL_HOME and $YULUP_WEBSITE .
  • Download the Yulup Website from Wyona's repository using subversion:
    svn co http://svn.wyona.com/repos/public/yulup/website
  • To add a new realm, we need to edit some properties files placed in $YANEL_HOME/conf. To do this, we will copy some *.properties files to local.*.properties.
  • Copy the file yanel-rti-yarep-properties to local.yanel-rti-yarep.properties and add the following entry to the configurations property
    # Relative Paths
    configurations=yulup,$YULUP_WEBSITE/yulup-website/yanel/config/yanel-rti-repository.xml
  • Copy the file yarep.properties to local.yarep.properties and add the following entry to the configurations property
    #Relative Paths
    configurations=yulup,$YULUP_WEBSITE/yulup-website/yanel/config/data-repository.xml
  • Copy the file ac-identities-yarep.properties to local.ac-identities-yarep.properties and add the following entry to the configurations property
    #Relative Paths configurations=yulup,$YULUP_WEBSITE/yulup-website/yanel/config/ac-identities-repository.xml
  • Copy the file ac-policies-yarep.properties to local.ac-policies-yarep.properties and add the following entry to the configurations property
    #Relative Paths
    configurations=yulup,$YULUP_WEBSITE/yulup-website/yanel/config/ac-policies-repository.xml
  • Copy the file realms.xml to local.realms.xml and add the following entry
    <realm id="yulup" mount-point="/yulup/">
    <name>Yulup Website</name>
    </realm>
