Getting Started

Prerequisites

Java Development Kit version 1.4.2 or higher (http://java.sun.com, e.g. J2SE SDK v 1.4.2_12)

Download Yanel

Download Yanel either as source or as binary.

Install Yanel

Install the binary:

  • Unpack the zip file
  • Run install
  • Startup Yanel

Install from source:

  • configure
  • build
  • yanel (start/stop)

Use Yanel

Browse to http://127.0.0.1:8888 and use Yanel either as author, integrator or developer.
