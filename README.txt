

    Y A N E L (Yet Another Lenya)
    =============================


    Prerequisites
    -------------

    Java Development Kit version 1.4.2 or higher


    Getting Started
    ---------------

    1) configure
    2) build
    3) yanel
    4) http://127.0.0.1:8888


    Principles
    ----------

    - Continuous Integration/Testing
    - Performance
    - Scalability
    - Redundancy/Clusterable
    - Forward and Backwards Compatibility
      - See http://www.ftponline.com/ea/magazine/winter2006/features/akrapf/default_pf.aspx
      - See http://davidbau.com/archives/2003/12/01/theory_of_compatibility_part_1.html
      - See http://www-128.ibm.com/developerworks/webservices/library/ws-soa-backcomp/
      - See http://cocoon.apache.org/versioning.html
    - Upgradeability
    - Out-of-box usage (minimal setup)
    - Package manager
    - Security
    - I18n/Localization
    - Integrated Documentation
    - Logging/Debugging during production
    - Rapid prototyping/development
    - Component based/Modularity
    - Release Planning/Roadmap
